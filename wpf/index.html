Macabre2D.Wpf

Currently, the two programs that interact with Macabre2D.Framework (Macabre2D.Editor and Cosmic Jam) are built using WPF. This library provides common controls, converters, and resources for those two programs. This also features a control that can run a Macabre2D game within a WPF interface.
