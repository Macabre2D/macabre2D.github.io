Macabresoft

Macabresoft is Brett Story. I make software under this label of sorts. Pretty much everything I develop is open-source, so check out the Macabresoft GitHub page to get into specifics.

In general, the main focus of Macabresoft is on games with the Macabre2D engine, but I also delve into audio programming and productivity tools. Follow @macabresoft for updates.

Macabre2D

A 2D game engine built on top of MonoGame. Custom features include physics, tile maps, sprite animations, audio playback with looping, and more.

Previously included a Windows only editor, but the entire engine is currently being reworked from the ground up to support a multi-platform editor using AvaloniaUI.

Piano Chord Catalog

A piano chord catalog that uses integer notation and math to display finger positions for various chords. The initial focus was common triad chords, but more may come.

This was a project to learn a little bit of angular. Sure, you could do something lame like read a book or go to some other website, or you could come look at piano chords on this beautiful web app.

Guitar Tuner

A guitar tuning application for desktop. Tune to standard guitar strings, drop D, and standard bass strings. The tuning can be set to automatically de nearest note, or the user can select which note they are tuning towards.

AvaloniaEx

A theme and control library for AvaloniaUI. Used by Macabre2D and the Guitar Tuner.

Macabresoft.Core

.NET Core libraries used across Macabresoft products.
