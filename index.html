• Macabre2D.Framework

    As the core of the entire suite, Macabre2D.Framework provides all the basic necessities to run games and tools developed with Macabre2D. The framework includes game components, modules, definitions for consumed assets, the game loop, rendering, physics, a synthesizer, and more. All other parts of the suite merely provide ways to interact with the framework. Along with the framework, this project also contains unit tests and example/test projects.

  • Macabre2D.Editor

    Macabre2D.Editor is a WPF program that allows users to create games for the Macabre2D.Framework. It features a scene editor, asset management, and the configuration of game settings for your progress. The editor is currently highly volatile and may have breaking changes with serialization regularly, so it is recommended to not use it for anything other than hobbyist development.

  • Cosmic Jam

    Cosmic Jam is an interface to create songs for the sythesizer within Macabre2D.Framework. The project is currently a work in progress and the UI exists mostly for messing around and not for serious song creation. More to come eventually.

    • View Source

  • Macabre2D.Wpf

    Currently, the two programs that interact with Macabre2D.Framework (Macabre2D.Editor and Cosmic Jam) are built using WPF. This library provides common controls, converters, and resources for those two programs. This also features a control that can run a Macabre2D game within a WPF interface.
