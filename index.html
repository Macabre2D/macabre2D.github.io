• Macabre2D

    Macabre2D is a suite of 2D game development tools. I began this project as frustration mounted when using other engines that could technically make 2D games, but had a far larger focus on 3D. Macabre2D aims to simplify cross-platform 2D game development.
    • Macabre2D.Framework
    • Macabre2D.Editor
    • Macabre2D.Wpf

  • Cosmic Jam

    Cosmic Jam is an interface to create songs for the sythesizer within Macabre2D.Framework. The project is currently a work in progress and the UI exists mostly for messing around and not for serious song creation. More to come.
